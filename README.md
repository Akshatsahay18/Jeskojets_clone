# Jesko Jets Clone

Cinematic luxury aviation experience built with Next.js 14, TypeScript, Tailwind CSS, Framer Motion, Lenis, and canvas-based image sequences.

## Stack

- Next.js 14 App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Lenis
- HTML5 Canvas

## Local Run

```bash
npm install
npm run build
npm run start -- --hostname 127.0.0.1 --port 3000
```
